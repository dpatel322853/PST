
# PST Screening Tool v2

Run locally:

```bash
pip install -r requirements_v2.txt
streamlit run streamlit_pst_tool_v2.py
```

Files:
- `pst_model_v2.py`: core model, PR EOS properties, scenario manager
- `report_export_v2.py`: Excel, Word, PDF export
- `streamlit_pst_tool_v2.py`: Streamlit UI

Important:
- Screening / engineering estimation only.
- Not for final design without validation.
